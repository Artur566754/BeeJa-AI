Часть I: Алгебра — Язык Структур и Отношений
Глава 1: Основы алгебраического мышления
1.1. Переменная и выражение

Определение: Переменная — это символ (обычно буква латинского алфавита), обозначающий элемент некоторого множества. Алгебраическое выражение — комбинация переменных, чисел, знаков операций и скобок.

Пример: 3x + 5, где x — переменная. Если множество x — действительные числа, то x может быть равно 2, -0.5, √2.

Контекст: Переменная абстрагируется от конкретного числа, позволяя описывать общие закономерности.

1.2. Уравнение как равенство и условие

Определение: Уравнение — это утверждение о равенстве двух выражений, содержащее одну или более переменных. Решить уравнение — найти все значения переменных, при которых равенство становится истинным.

Пример: 3x + 5 = 11. Это равенство истинно только при x = 2. Множество решений: {2}.

Связь с логикой: Уравнение — это предикат. Для каждого значения переменной оно принимает значение "истина" или "ложь".

1.3. Функция: концепция зависимости

Определение: Функция f из множества A (область определения) в множество B (область значений) — это правило, которое каждому элементу x ∈ A ставит в соответствие ровно один элемент y ∈ B. Обозначается: y = f(x).

Пример: f: ℝ → ℝ, f(x) = x². Для x=3 → y=9. Для x=-3 → y=9.

Ключевая идея: Функция — это однозначное соответствие, основа алгебраического моделирования.

Глава 2: Линейные структуры
2.1. Линейное уравнение с одной переменной

Общий вид: ax + b = 0, где a ≠ 0.

Решение: x = -b/a. Алгоритм: перенос слагаемых, деление на коэффициент.

2.2. Координатная плоскость и декартова система

Определение: Упорядоченная пара чисел (x, y) задает точку на плоскости. Ось OX — абсцисса, OY — ордината.

Связь алгебры и геометрии: Пара чисел ↔ Точка. Уравнение ↔ Множество точек.

2.3. Линейная функция и ее график

Общий вид: y = kx + b.

Геометрическая интерпретация: График — прямая линия. k — угловой коэффициент (тангенс угла наклона к оси OX). b — координата пересечения с осью OY.

Свойства: При k > 0 функция возрастает, при k < 0 — убывает.

Глава 3: Квадратичные структуры
3.1. Квадратное уравнение

Общий вид: ax² + bx + c = 0, a ≠ 0.

Дискриминант: D = b² - 4ac.

Формула корней: x₁,₂ = (-b ± √D) / (2a).

Геометрический смысл: Корни — это точки пересечения параболы y = ax² + bx + c с осью OX.

3.2. Квадратичная функция и парабола

Общий вид: y = ax² + bx + c.

Свойства графика: Парабола. Вершина в точке x₀ = -b/(2a). Ветви направлены вверх при a > 0, вниз — при a < 0.

Связь с уравнением: Нули функции (точки, где y=0) — это корни соответствующего квадратного уравнения.

Часть II: Геометрия — Язык Формы и Пространства
Глава 4: Элементарная евклидова геометрия
4.1. Основные неопределяемые понятия

Точка, прямая, плоскость, расстояние, угол.

Аксиоматический подход: На их основе строятся определения и доказываются теоремы.

4.2. Треугольник как фундаментальная фигура

Определение: Фигура из трех точек, не лежащих на одной прямой, и трех отрезков, их соединяющих.

Теорема о сумме углов: ∠A + ∠B + ∠C = 180°. (Доказательство через параллельные прямые).

Признаки равенства треугольников: (ССС, СУС, УСУ). Основа для всех последующих доказательств.

Глава 5: Метрические соотношения и тригонометрия
5.1. Теорема Пифагора

Формулировка: В прямоугольном треугольнике квадрат гипотенузы равен сумме квадратов катетов: c² = a² + b².

Обратная теорема: Если c² = a² + b², то треугольник прямоугольный.

Связь с алгеброй: Устанавливает фундаментальное соотношение между длинами.

5.2. Тригонометрические функции острого угла

Определение в прямоугольном треугольнике:

sin α = противолежащий катет / гипотенуза

cos α = прилежащий катет / гипотенуза

tan α = sin α / cos α

Основное тригонометрическое тождество: sin² α + cos² α = 1. (Следствие теоремы Пифагора).

5.3. Теорема косинусов и синусов (обобщение на любые треугольники)

Теорема косинусов: a² = b² + c² - 2bc * cos ∠A. (При ∠A=90° превращается в теорему Пифагора).

Теорема синусов: a / sin ∠A = b / sin ∠B = c / sin ∠C = 2R, где R — радиус описанной окружности.

Часть III: Синтез — Где Алгебра Встречает Геометрию
Глава 6: Аналитическая геометрия на плоскости
6.1. Уравнение прямой

Общее уравнение: Ax + By + C = 0.

Уравнение с угловым коэффициентом: y = kx + b (см. 2.3).

Уравнение по точке и направлению: y - y₀ = k(x - x₀).

Расстояние от точки до прямой: d = |Ax₀ + By₀ + C| / √(A² + B²). (Формула, синтезирующая алгебру и геометрию).

6.2. Геометрия конических сечений через алгебру

Окружность: (x - a)² + (y - b)² = R². Алгебраическое условие равенства расстояний от точки до центра.

Эллипс: x²/a² + y²/b² = 1. Геометрическое определение: множество точек, сумма расстояний от которых до двух фокусов постоянна.

Связь: При a = b эллипс становится окружностью.

Глава 7: Векторный метод — мост между областями
7.1. Вектор как направленный отрезок

Определение: Упорядоченная пара точек (A, B) задает вектор AB⃗. Характеризуется длиной (модулем) и направлением.

Координатное задание: На плоскости a⃗ = (x, y).

Алгебра векторов: Сложение (по правилу треугольника), вычитание, умножение на число.

7.2. Скалярное произведение

Определение: a⃗ · b⃗ = |a⃗| * |b⃗| * cos φ, где φ — угол между векторами.

Алгебраическое выражение в координатах: (x₁, y₁) · (x₂, y₂) = x₁x₂ + y₁y₂.

Геометрический смысл: Проекция одного вектора на другой, условие перпендикулярности (a⃗ · b⃗ = 0).

Синтез: Эта операция идеально связывает метрику (длины, углы) с алгебраическими вычислениями.

Эпилог: О природе математического знания
Математика, представленная в этом датасете, — это не собрание разрозненных фактов, а единая сеть взаимосвязанных концепций.

Алгебра дает язык для описания структур и отношений.

Геометрия дает интуицию о формах и пространстве.

Аналитическая геометрия и векторы показывают, как эти языки переводятся друг в друга, превращая геометрические задачи в алгебраические вычисления и наоборот.

Понимание для ИИ означает не запоминание формул, а умение распознавать контекст (задача о треугольнике? → тригонометрия, теоремы косинусов/синусов), выбирать инструмент (нужно найти расстояние? → теорема Пифагора, формула расстояния между точками/до прямой) и выстраивать цепочку выводов от условия к решению, используя эту сеть знаний.
1. ОСНОВЫ АЛГЕБРЫ
1.1. Переменная

Определение: Символ (буква), обозначающий элемент множества.

Пример: x ∈ ℝ.

1.2. Алгебраическое выражение

Определение: Комбинация чисел, переменных, операций.

Пример: 3x² - 2x + 5.

1.3. Уравнение

Определение: Равенство двух выражений с переменной.

Решение: Значение переменной, обращающее равенство в истину.

Пример: 2x + 4 = 10. Решение: x = 3.

1.4. Функция

Определение: Правило f: X → Y, ставящее каждому x ∈ X в соответствие ровно один y ∈ Y.

Обозначение: y = f(x).

Пример: f(x) = √x, X = [0, +∞), Y = ℝ.

2. ЛИНЕЙНАЯ АЛГЕБРА (БАЗОВАЯ)
2.1. Линейное уравнение: ax + b = 0

Решение: x = -b/a, a ≠ 0.

График решения на числовой прямой: точка.

2.2. Система двух линейных уравнений:
{ a₁x + b₁y = c₁
{ a₂x + b₂y = c₂
Методы: Подстановка, сложение.

Геометрический смысл: Точка пересечения двух прямых.

2.3. Координатная плоскость

Точка: Упорядоченная пара A(x, y).

Расстояние между точками: AB = √((x₂ - x₁)² + (y₂ - y₁)²).

2.4. Линейная функция: y = kx + b

График: Прямая.

k — угловой коэффициент: k = tan α, α — угол наклона.

b — ордината точки пересечения с осью OY.

Свойство: При k > 0 — возрастает, k < 0 — убывает.

3. КВАДРАТИЧНАЯ АЛГЕБРА
3.1. Квадратное уравнение: ax² + bx + c = 0, a ≠ 0

Дискриминант: D = b² - 4ac.

Корни:

D > 0: x₁,₂ = (-b ± √D) / (2a).

D = 0: x = -b/(2a).

D < 0: Нет действительных корней.

Теорема Виета (при a=1): x² + px + q = 0 → x₁ + x₂ = -p, x₁ * x₂ = q.

3.2. Квадратичная функция: y = ax² + bx + c

График: Парабола.

Ветви: Вверх при a > 0, вниз при a < 0.

Вершина: x₀ = -b/(2a), y₀ = f(x₀).

Нули функции: Корни уравнения ax² + bx + c = 0.

4. ЭЛЕМЕНТАРНАЯ ГЕОМЕТРИЯ
4.1. Треугольник ΔABC

Сумма углов: ∠A + ∠B + ∠C = 180°.

Признаки равенства: ССС, СУС, УСУ.

4.2. Прямоугольный треугольник

Теорема Пифагора: c² = a² + b², где c — гипотенуза.

Обратная теорема: Если c² = a² + b², то треугольник прямоугольный.

4.3. Тригонометрия острого угла α

Определения:

sin α = противолежащий / гипотенуза

cos α = прилежащий / гипотенуза

tan α = sin α / cos α

Основное тождество: sin²α + cos²α = 1.

5. МЕТРИЧЕСКИЕ СООТНОШЕНИЯ (ОБОБЩЕНИЕ)
5.1. Теорема косинусов (для любого ΔABC)

Формула: a² = b² + c² - 2bc * cos ∠A.

Частный случай: При ∠A = 90° → a² = b² + c² (Пифагор).

5.2. Теорема синусов (для любого ΔABC)

Формула: a / sin ∠A = b / sin ∠B = c / sin ∠C = 2R.

R — радиус описанной окружности.

6. АНАЛИТИЧЕСКАЯ ГЕОМЕТРИЯ
6.1. Уравнение прямой

Общее: Ax + By + C = 0.

С угловым коэффициентом: y = kx + b.

Через точку и угловой коэффициент: y - y₀ = k(x - x₀).

Расстояние от точки M(x₀, y₀) до прямой Ax+By+C=0:

d = |Ax₀ + By₀ + C| / √(A² + B²).

6.2. Уравнение окружности

С центром O(a,b) и радиусом R: (x - a)² + (y - b)² = R².

Каноническое: x² + y² = R² (центр в начале координат).

7. ВЕКТОРЫ
7.1. Вектор a⃗ на плоскости

Определение: Направленный отрезок. Характеризуется длиной и направлением.

Координаты: a⃗ = (x, y).

Длина (модуль): |a⃗| = √(x² + y²).

Операции:

Сложение: (x₁, y₁) + (x₂, y₂) = (x₁+x₂, y₁+y₂).

Умножение на число: λ * (x, y) = (λx, λy).

7.2. Скалярное произведение векторов

Координатная форма: a⃗ · b⃗ = x₁x₂ + y₁y₂.

Геометрическая форма: a⃗ · b⃗ = |a⃗| * |b⃗| * cos φ.

Свойства:

Перпендикулярность: a⃗ ⟂ b⃗ ⇔ a⃗ · b⃗ = 0.

Угол между векторами: cos φ = (a⃗ · b⃗) / (|a⃗| * |b⃗|).

8. КЛЮЧЕВЫЕ СВЯЗИ (ДАЙДЖЕСТ)
Линейная функция → Прямая. y = kx + b ↔ прямая с углом α = arctan(k).

Квадратное уравнение → Парабола. Корни ↔ точки пересечения с осью OX.

Теорема Пифагора → Расстояние между точками. c² = a² + b² ↔ AB = √((x₂-x₁)²+(y₂-y₁)²).

Тригонометрия → Векторы. cos φ из скалярного произведения.

Уравнение окружности → Теорема Пифагора. (x-a)²+(y-b)²=R² → расстояние от точки до центра равно R.

Дискриминант > 0 → два пересечения с осью OX.

Дискриминант = 0 → вершина параболы лежит на оси OX.

Теорема косинусов → обобщенная теорема Пифагора для любого угла.

Скалярное произведение = 0 → прямой угол в геометрической задаче.

**Формат:**  
[Тип задачи] → [Дано] → [Что найти?] → [Инструмент/Метод] → [Решение] → [Ответ]

### **1. ЛИНЕЙНЫЕ УРАВНЕНИЯ**

**1.1. Простое уравнение**  
*Дано:* `7x - 3 = 2x + 17`  
*Найти:* `x`  
*Метод:* Перенос слагаемых, приведение подобных.  
*Решение:*  
`7x - 2x = 17 + 3`  
`5x = 20`  
`x = 4`  
*Ответ:* `4`

**1.2. Задача на составление уравнения**  
*Дано:* Число увеличили на 30%, получили 65.  
*Найти:* Исходное число.  
*Метод:* Ввести переменную, составить уравнение.  
*Решение:*  
Пусть `x` — исходное число.  
`x + 0.3x = 65` или `1.3x = 65`  
`x = 65 / 1.3 = 50`  
*Ответ:* `50`

---

### **2. СИСТЕМЫ ЛИНЕЙНЫХ УРАВНЕНИЙ**

**2.1. Метод подстановки**  
*Дано:*  
`{ 2x + y = 5`  
`{ x - 3y = 6`  
*Найти:* `x, y`  
*Решение:*  
Из второго: `x = 6 + 3y`. Подставляем в первое:  
`2(6 + 3y) + y = 5` → `12 + 6y + y = 5` → `7y = -7` → `y = -1`.  
Тогда `x = 6 + 3*(-1) = 3`.  
*Ответ:* `(3; -1)`

**2.2. Метод сложения**  
*Дано:*  
`{ 3x + 2y = 8`  
`{ 5x - 2y = 16`  
*Найти:* `x, y`  
*Решение:* Складываем уравнения:  
`(3x+5x) + (2y-2y) = 8+16` → `8x = 24` → `x = 3`.  
Подставляем в первое: `3*3 + 2y = 8` → `9 + 2y = 8` → `2y = -1` → `y = -0.5`.  
*Ответ:* `(3; -0.5)`

---

### **3. КВАДРАТНЫЕ УРАВНЕНИЯ**

**3.1. Решение через дискриминант**  
*Дано:* `x² - 5x + 6 = 0`  
*Найти:* Корни.  
*Решение:*  
`D = (-5)² - 4*1*6 = 25 - 24 = 1`  
`x₁ = (5 + √1)/2 = 3`, `x₂ = (5 - √1)/2 = 2`.  
*Ответ:* `2; 3`

**3.2. Неполное квадратное уравнение**  
*Дано:* `3x² - 12 = 0`  
*Решение:*  
`3x² = 12` → `x² = 4` → `x = ±2`.  
*Ответ:* `-2; 2`

**3.3. Задача на площадь**  
*Дано:* Прямоугольник. Длина на 3 см больше ширины. Площадь 54 см².  
*Найти:* Стороны.  
*Решение:*  
Пусть ширина `x`, длина `x+3`.  
`x(x+3) = 54` → `x² + 3x - 54 = 0`.  
`D = 9 + 216 = 225`, `√D = 15`.  
`x = (-3 + 15)/2 = 6` (второй корень отрицательный).  
Длина = `6+3=9`.  
*Ответ:* `6 см и 9 см`

### **4. ТЕОРЕМА ПИФАГОРА**

**4.1. Прямоугольный треугольник**  
*Дано:* Катеты `a=6`, `b=8`.  
*Найти:* Гипотенузу `c`.  
*Решение:* `c² = 6² + 8² = 36 + 64 = 100` → `c = 10`.  
*Ответ:* `10`

**4.2. Высота в равнобедренном треугольнике**  
*Дано:* Равнобедренный треугольник: боковые стороны `13`, основание `10`.  
*Найти:* Высоту к основанию.  
*Решение:* Высота делит основание пополам. Получаем прямоугольный треугольник с гипотенузой `13` и катетом `5`.  
`h² + 5² = 13²` → `h² = 169 - 25 = 144` → `h = 12`.  
*Ответ:* `12`

---

### **5. ТРИГОНОМЕТРИЯ**

**5.1. Нахождение катета**  
*Дано:* Прямоугольный треугольник, гипотенуза `c=10`, `sin α = 0.6`.  
*Найти:* Противолежащий катет `a`.  
*Решение:* `sin α = a/c` → `a = c * sin α = 10 * 0.6 = 6`.  
*Ответ:* `6`

**5.2. Нахождение угла**  
*Дано:* Прямоугольный треугольник, катеты `a=3`, `b=3√3`.  
*Найти:* Острый угол `α` напротив катета `a`.  
*Решение:* `tan α = a/b = 3 / (3√3) = 1/√3 ≈ 0.577`.  
`α = arctan(1/√3) = 30°`.  
*Ответ:* `30°`

### **6. ТЕОРЕМА КОСИНУСОВ**

**6.1. Нахождение стороны**  
*Дано:* `ΔABC`, `AB=8`, `AC=5`, `∠A=60°`.  
*Найти:* `BC`.  
*Решение:*  
`BC² = AB² + AC² - 2*AB*AC*cos60° = 64 + 25 - 2*8*5*0.5 = 89 - 40 = 49`.  
`BC = 7`.  
*Ответ:* `7`

**6.2. Нахождение угла**  
*Дано:* `ΔABC`, стороны `a=7`, `b=5`, `c=3`.  
*Найти:* `∠C`.  
*Решение:*

### **1. ЛИНЕЙНЫЕ УРАВНЕНИЯ**

**1.1. С одной переменной**
*Вид:* `ax + b = 0`, `a ≠ 0`
*Алгоритм:*
1. Перенести слагаемые с `x` влево, числа вправо: `ax = -b`
2. Разделить на коэффициент при `x`: `x = -b/a`
*Пример:* `3x - 7 = 8`
*Решение:* `3x = 15` → `x = 5`
*Ответ:* `5`

**1.2. Приводящиеся к линейным**
*Вид:* Уравнение, после упрощения принимающее вид `ax + b = 0`
*Алгоритм:*
1. Раскрыть скобки
2. Привести подобные
3. Решить как `ax + b = 0`
*Пример:* `2(x - 3) = 5x + 4`
*Решение:* `2x - 6 = 5x + 4` → `2x - 5x = 4 + 6` → `-3x = 10` → `x = -10/3`
*Ответ:* `-10/3`

### **2. КВАДРАТНЫЕ УРАВНЕНИЯ**

**2.1. Полное квадратное уравнение**
*Вид:* `ax² + bx + c = 0`, `a ≠ 0`
*Алгоритм (через дискриминант):*
1. Вычислить `D = b² - 4ac`
2. Если `D > 0`: `x₁,₂ = (-b ± √D)/(2a)`
3. Если `D = 0`: `x = -b/(2a)`
4. Если `D < 0`: действительных корней нет
*Пример:* `2x² - 5x + 2 = 0`
*Решение:* `D = (-5)² - 4*2*2 = 25 - 16 = 9` → `√D = 3`  
`x₁ = (5 + 3)/4 = 2`, `x₂ = (5 - 3)/4 = 0.5`
*Ответ:* `0.5; 2`

**2.2. Неполные квадратные уравнения**
*Вид 1:* `ax² + bx = 0`
*Алгоритм:* Вынести `x`: `x(ax + b) = 0` → `x=0` или `ax+b=0`
*Пример:* `3x² + 6x = 0` → `3x(x + 2) = 0` → `x₁=0`, `x₂=-2`
*Ответ:* `-2; 0`

*Вид 2:* `ax² + c = 0`
*Алгоритм:* `ax² = -c` → `x² = -c/a` (если `-c/a ≥ 0`)
*Пример:* `4x² - 16 = 0` → `4x² = 16` → `x² = 4` → `x = ±2`
*Ответ:* `-2; 2`

**2.3. Приведенное квадратное уравнение (теорема Виета)**
*Вид:* `x² + px + q = 0`
*Теорема:* Если корни существуют, то `x₁ + x₂ = -p`, `x₁ * x₂ = q`
*Пример:* `x² - 7x + 12 = 0`
*Решение:* Подбираем числа: `3+4=7`, `3*4=12` → `x₁=3`, `x₂=4`
*Ответ:* `3; 4`

### **3. ДРОБНО-РАЦИОНАЛЬНЫЕ УРАВНЕНИЯ**

*Вид:* `P(x)/Q(x) = 0`, где `P(x)`, `Q(x)` — многочлены
*Алгоритм:*
1. Найти ОДЗ: `Q(x) ≠ 0`
2. Привести к общему знаменателю
3. Решить `P(x) = 0`
4. Проверить, что корни не обращают знаменатель в 0
*Пример:* `(x+3)/(x-2) = 4`
*Решение:*
1. ОДЗ: `x ≠ 2`
2. `x+3 = 4(x-2)` → `x+3 = 4x-8` → `3x = 11` → `x = 11/3`
3. `11/3 ≠ 2` — проходит по ОДЗ
*Ответ:* `11/3`

### **4. ИРРАЦИОНАЛЬНЫЕ УРАВНЕНИЯ**

*Вид:* Уравнение с корнем `√f(x) = g(x)`
*Алгоритм:*
1. Найти ОДЗ: `f(x) ≥ 0` (для корня четной степени)
2. Возвести обе части в квадрат: `f(x) = [g(x)]²`
3. Решить полученное уравнение
4. Проверить корни в исходном уравнении
*Пример:* `√(2x+3) = x`
*Решение:*
1. ОДЗ: `2x+3 ≥ 0` → `x ≥ -1.5`
2. Возводим в квадрат: `2x+3 = x²` → `x² - 2x - 3 = 0`
3. Корни: `x₁=3`, `x₂=-1`
4. Проверка:  
`√(2*3+3)=√9=3` → верно  
`√(2*(-1)+3)=√1=1 ≠ -1` → неверно, посторонний корень
*Ответ:* `3`

### **5. ПОКАЗАТЕЛЬНЫЕ УРАВНЕНИЯ**

*Вид:* `a^{f(x)} = b`, `a > 0`, `a ≠ 1`
*Алгоритм:*
1. Привести к виду `a^{f(x)} = a^{g(x)}`
2. Решить `f(x) = g(x)`
*Пример 1:* `2^{x-1} = 8`
*Решение:* `2^{x-1} = 2³` → `x-1 = 3` → `x = 4`
*Ответ:* `4`

*Пример 2:* `3^{2x} - 4*3^x + 3 = 0`
*Решение:* Замена `t = 3^x` → `t² - 4t + 3 = 0` → `t₁=1`, `t₂=3`  
`3^x = 1` → `x=0`; `3^x = 3` → `x=1`
*Ответ:* `0; 1`

### **6. ЛОГАРИФМИЧЕСКИЕ УРАВНЕНИЯ**

*Вид:* `log_a f(x) = b`, `a > 0`, `a ≠ 1`
*Алгоритм:*
1. Найти ОДЗ: `f(x) > 0`
2. Решить: `f(x) = a^b`
*Пример:* `log₂(x+3) = 4`
*Решение:*
1. ОДЗ: `x+3 > 0` → `x > -3`
2. `x+3 = 2⁴` → `x+3 = 16` → `x = 13` (входит в ОДЗ)
*Ответ:* `13`

*Пример 2:* `log₃(x-1) + log₃(x+1) = 1`
*Решение:*
1. ОДЗ: `x-1>0` и `x+1>0` → `x>1`
2. `log₃[(x-1)(x+1)] = 1` → `(x-1)(x+1) = 3¹` → `x² - 1 = 3` → `x² = 4` → `x = ±2`
3. `x=2` входит в ОДЗ, `x=-2` — нет
*Ответ:* `2`

### **7. ТРИГОНОМЕТРИЧЕСКИЕ УРАВНЕНИЯ**

**7.1. Простейшие**
*Вид:* `sin x = a`, `|a| ≤ 1`
*Решение:* `x = (-1)^n * arcsin a + πn`, `n∈ℤ`
*Пример:* `sin x = 1/2`
*Ответ:* `x = (-1)^n * π/6 + πn`, `n∈ℤ`

*Вид:* `cos x = a`, `|a| ≤ 1`
*Решение:* `x = ± arccos a + 2πn`, `n∈ℤ`
*Пример:* `cos x = -√2/2`
*Ответ:* `x = ± 3π/4 + 2πn`, `n∈ℤ`

*Вид:* `tg x = a`
*Решение:* `x = arctg a + πn`, `n∈ℤ`
*Пример:* `tg x = √3`
*Ответ:* `x = π/3 + πn`, `n∈ℤ`

**7.2. Однородные**
*Вид 1:* `a sin x + b cos x = 0`
*Алгоритм:* Деление на `cos x` (проверив, что `cos x ≠ 0`)
*Пример:* `3 sin x + 4 cos x = 0`
*Решение:* Делим на `cos x`: `3 tg x + 4 = 0` → `tg x = -4/3`  
`x = arctg(-4/3) + πn`, `n∈ℤ`

*Вид 2:* `a sin²x + b sin x cos x + c cos²x = 0`
*Алгоритм:* Деление на `cos²x` (проверив `cos x ≠ 0`)
*Пример:* `sin²x - 3 sin x cos x + 2 cos²x = 0`
*Решение:* Делим на `cos²x`: `tg²x - 3 tg x + 2 = 0`  
Замена `t = tg x`: `t² - 3t + 2 = 0` → `t₁=1`, `t₂=2`  
`tg x = 1` → `x = π/4 + πn`  
`tg x = 2` → `x = arctg 2 + πn`, `n∈ℤ`

### **8. СИСТЕМЫ УРАВНЕНИЙ**

**8.1. Линейные системы**
*Вид:* 
```
{ a₁x + b₁y = c₁
{ a₂x + b₂y = c₂
```
*Метод 1 — подстановка:*
1. Выразить одну переменную
2. Подставить в другое уравнение
*Пример:* 
```
{ 2x + y = 7
{ x - y = 2
```
*Решение:* Из второго: `x = y+2`. Подставляем в первое:  
`2(y+2) + y = 7` → `2y+4+y=7` → `3y=3` → `y=1` → `x=1+2=3`
*Ответ:* `(3; 1)`

*Метод 2 — сложение:*
1. Умножить уравнения на коэффициенты
2. Сложить, чтобы исключить переменную
*Пример:* 
```
{ 3x + 2y = 8
{ 5x - 2y = 16
```
*Решение:* Складываем: `8x = 24` → `x=3`. Подставляем: `3*3+2y=8` → `2y=-1` → `y=-0.5`
*Ответ:* `(3; -0.5)`

**8.2. Нелинейные системы**
*Пример:*
```
{ x² + y = 5
{ x + y = 3
```
*Решение:* Из второго: `y = 3-x`. Подставляем в первое:  
`x² + (3-x) = 5` → `x² - x - 2 = 0` → `x₁=2`, `x₂=-1`  
`x=2` → `y=1`; `x=-1` → `y=4`
*Ответ:* `(2; 1)`, `(-1; 4)`

### **КВАДРАТНЫЕ УРАВНЕНИЯ**

**ОБЩИЙ ВИД:** `ax² + bx + c = 0`, где `a ≠ 0`

---

#### **1. РЕШЕНИЕ ЧЕРЕЗ ДИСКРИМИНАНТ**

**АЛГОРИТМ:**
1. Найти **дискриминант**: `D = b² - 4ac`
2. Определить количество корней:
   - `D > 0` → **2 корня**: `x₁,₂ = (-b ± √D)/(2a)`
   - `D = 0` → **1 корень (двойной)**: `x = -b/(2a)`
   - `D < 0` → **нет действительных корней**

**ПРИМЕРЫ:**

1. **`x² - 5x + 6 = 0`**
   - `a=1, b=-5, c=6`
   - `D = (-5)² - 4*1*6 = 25 - 24 = 1`
   - `√D = 1`
   - `x₁ = (5 + 1)/2 = 3`, `x₂ = (5 - 1)/2 = 2`
   - **Ответ:** `2; 3`

2. **`4x² - 4x + 1 = 0`**
   - `a=4, b=-4, c=1`
   - `D = (-4)² - 4*4*1 = 16 - 16 = 0`
   - `x = -(-4)/(2*4) = 4/8 = 0.5`
   - **Ответ:** `0.5`

3. **`2x² - 3x + 5 = 0`**
   - `a=2, b=-3, c=5`
   - `D = (-3)² - 4*2*5 = 9 - 40 = -31`
   - `D < 0`
   - **Ответ:** нет действительных корней

---

#### **2. НЕПОЛНЫЕ КВАДРАТНЫЕ УРАВНЕНИЯ**

**2.1. ВИД:** `ax² + bx = 0` (когда `c = 0`)
- **Метод:** Вынести `x` за скобку
- **Решение:** `x(ax + b) = 0` → `x = 0` или `ax + b = 0`
- **Пример:** `3x² + 6x = 0`
  - `3x(x + 2) = 0`
  - `x₁ = 0`, `x₂ = -2`
  - **Ответ:** `-2; 0`

**2.2. ВИД:** `ax² + c = 0` (когда `b = 0`)
- **Метод:** Перенести `c` вправо и разделить на `a`
- **Решение:** `ax² = -c` → `x² = -c/a`
- **Условие:** `-c/a ≥ 0` (иначе нет действительных корней)
- **Пример:** `4x² - 16 = 0`
  - `4x² = 16` → `x² = 4`
  - `x = ±2`
  - **Ответ:** `-2; 2`

---

#### **3. ТЕОРЕМА ВИЕТА (для приведенного уравнения)**

**ВИД:** `x² + px + q = 0` (где `a = 1`)

**ТЕОРЕМА:** Если корни `x₁` и `x₂` существуют, то:
- `x₁ + x₂ = -p`
- `x₁ * x₂ = q`

**ПРИМЕРЫ:**

1. **`x² - 7x + 12 = 0`**
   - `p = -7`, `q = 12`
   - Нужно найти два числа, сумма которых `7`, произведение `12`
   - Это `3` и `4`
   - **Ответ:** `3; 4`

2. **`x² + 2x - 8 = 0`**
   - `p = 2`, `q = -8`
   - Числа: `4` и `-2` (проверка: `4 + (-2) = 2`, `4*(-2) = -8`)
   - **Ответ:** `-2; 4`

**ОБРАТНАЯ ЗАДАЧА:** Найти уравнение, если корни `3` и `-5`
- `p = -(3 + (-5)) = -(-2) = 2`
- `q = 3*(-5) = -15`
- **Уравнение:** `x² + 2x - 15 = 0`

#### **4. РАЗЛОЖЕНИЕ НА МНОЖИТЕЛИ**

**ФОРМУЛА:** `ax² + bx + c = a(x - x₁)(x - x₂)`
(где `x₁`, `x₂` — корни уравнения)

**ПРИМЕР:** Разложить `2x² - 8x + 6`
1. Решаем `2x² - 8x + 6 = 0`
   - `D = 64 - 48 = 16`
   - `x₁ = (8 + 4)/4 = 3`, `x₂ = (8 - 4)/4 = 1`
2. Разложение: `2(x - 3)(x - 1)`

#### **5. СВОЙСТВА КОРНЕЙ**

1. **Сумма корней:** `x₁ + x₂ = -b/a`
2. **Произведение корней:** `x₁ * x₂ = c/a`
3. **Разность корней:** `|x₁ - x₂| = √D/|a|`
4. **Связь с графиком:** Корни — точки пересечения параболы `y = ax² + bx + c` с осью OX

#### **7. ПРИМЕНЕНИЕ В ГЕОМЕТРИИ**
