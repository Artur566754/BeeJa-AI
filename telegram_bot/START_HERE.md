## ⚡ Самое главное (за 1 минуту)

### Что нужно загрузить на сервер:

```
/home/user/ai-telegram-bot/
├── telegram_bot/          (эта папка)
├── src/                   (папка с AI моделью)
└── main.py                (главный файл AI)
```

### Команды на сервере:

```bash
# 1. Перейди в папку
cd ai-telegram-bot/telegram_bot

# 2. Создай папки
mkdir -p datasets models bot_data

# 3. Скопируй датасеты
cp -r ../datasets/* ./datasets/

# 4. Запусти
docker-compose up -d

# 5. Проверь логи
docker-compose logs -f
```

### В Telegram:

1. Найди бота
2. `/start`
3. "🎓 Обучить модель" → `50`
4. Дождись завершения
5. "💬 Чат с AI"
