🚀 КАК ЗАГРУЗИТЬ:

Вариант 1: Архив
1. Создай ZIP архив с этими файлами
2. Загрузи на сервер через FileZilla
3. Распакуй на сервере: unzip ai-telegram-bot.zip

Вариант 2: Прямая загрузка
1. Подключись через FileZilla
2. Загрузи папки и файлы напрямую

Вариант 3: Git
1. Создай репозиторий
2. Закоммить файлы
3. На сервере: git clone

Вариант 4: SCP (командная строка)
scp -r telegram_bot/ src/ main.py datasets/ user@server:/path/
